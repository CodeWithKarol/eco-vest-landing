1Basic Setup

  • styles.css - All styling and responsive design
  • script.js - Interactive features and animations

2Changing Colors & Branding

Color Palette

Open styles.css and find the :root section at the top (around lines 1-20). Modify these CSS custom properties: 

:root {
    --primary-green: #2d7d5e;      /* Main brand color */
    --accent-green: #7ac8a0;       /* Accent highlights */
    --light-green: #5ab88d;        /* Lighter accent */
    --dark-text: #1a1a1a;          /* Headings */
    --text-secondary: #666666;     /* Body text */
    --bg-light: #f5f1e8;           /* Light backgrounds */
}
üí° Tip: Use a color picker tool or website like Coolors.co to generate a harmonious color palette that matches your brand.

7Typography Changes

Location: styles.css (around lines 30-50)

Changing Fonts

The template uses Inter and Poppins from Google Fonts. To use different fonts:

  1. Find the Google Fonts link in index.html head section
  2. Replace with your chosen fonts from Google Fonts
  3. Update CSS font-family declarations 
body {
    font-family: 'Your Font', sans-serif;
}

h1, h2, h3 {
    font-family: 'Your Heading Font', serif;
}

Font Sizes

The template uses clamp() for fluid typography. Example: 

.hero-title {
    font-size: clamp(1.75rem, 5vw, 3.5rem);
    /* min size, preferred size, max size */
}

10Animation & Interactions

}

13Troubleshooting

Common Issues

Issue: Changes not showing up

  • Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
  • Check if files are saved
  • Verify file paths are correct

Issue: Layout broken on mobile

  • Check for missing closing tags in HTML
  • Verify media queries in CSS
  • Use browser dev tools to inspect elements

Issue: Form not submitting

  • Check JavaScript console for errors (F12)
  • Verify form action URL is correct
  • Test with a simple alert to isolate the issue
